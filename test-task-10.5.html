Manual Testing Instructions:

Test the following interactions:

  1. Link Form Inputs: Hover over and click into the name and URL inputs - should show border color change and focus state
  2. Add Link Button: Hover over the "Add Link" button - should lift up and show shadow
  3. Add Link Button Click: Click the "Add Link" button - should show pressed state
  4. Link Buttons: Hover over link cards - should change color, lift up, and scale slightly
  5. Link Button Click: Click a link button - should show active state before navigation
  6. Delete Buttons: Hover over delete buttons - should lift up with red shadow
  7. Delete Button Click: Click a delete button - should show pressed state
  8. Visual Hierarchy: Verify link buttons are visually distinct as cards/pills
  9. Responsive Layout: Resize window to test form layout changes
