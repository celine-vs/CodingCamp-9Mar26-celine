Task 7.4 Functional Test - Quick Links Event Handlers

Test Instructions

This test verifies that Task 7.4 requirements are met:

  1. init() loads links and attaches event listeners
  2. Submit handler with preventDefault - Form submission should not reload page
  3. Click handlers for delete buttons - Delete buttons should remove links
  4. Clear input fields after successful creation - Inputs should be empty after adding
  5. Display validation errors for invalid URLs - Show error messages for bad input

Manual Test Steps:

  1. Add a valid link (e.g., "Google" and "https://google.com") - inputs should clear
  2. Try to add a link with empty name - should show validation error
  3. Try to add a link with empty URL - should show validation error
  4. Try to add a link with invalid URL (e.g., "google.com" without http://) - should show error
  5. Add multiple valid links - all should appear in the container
  6. Click delete button on a link - link should be removed
  7. Reload the page - links should persist from localStorage
